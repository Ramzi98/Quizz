L'application Quizzs est un jeu Android native construit avec le langage JAVA sur l’IDE Android Studio version 4.4.1 
conçu pour fonctionner sur la plate-forme Android.

Pour lancer l'application il faut suivre les étapes suivantes :

1)	D’abord il faut télécharger Android Studio depuis le site suivant : https://developer.android.com/studio 
	et Aussi les SDK aprés le lancement du programme.

2)	Ensuite on décompresse le fichier Zip de l’application par une application spécifique (Winrar par exemple).
	Après on va ouvrir Android Studio et on clique dans le menu sur File -> New -> Import Project et on sélectionne le projet depuis la machine.

3)	Le projet sera importé dans l’IDE on attend lorsque les gradles se charge.

4)	L’étape suivante et de brancher le téléphone mobile avec le pc via un câble USB et activer le mode développeur sur le
	le téléphone après activer le mode débogage.

5)	Et à la fin on appuie sur le Button run triangle vert dans le menu de Android Studio.

6) L'application sera lancée dans votre smartphone.
